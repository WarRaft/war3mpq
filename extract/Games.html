Game Types

Single Player Games

Starting Single-Player Campaigns

To start a single-player campaign game, follow these steps:

  1. Start Warcraft III: The Frozen Throne.

  2. Choose "Single Player".

  3. Enter a name for your Profile and choose "Create".

  4. Choose "Campaign".

  5. Next choose a difficulty level.

  6. Choose the first campaign mission to begin.




Battle.net Games

Joining and Creating Standard Ladder Games

To join or create a standard game, follow these steps:

NOTE: You must have an active Internet connection.

  1. Start Warcraft III: The Frozen Throne.

  2. Select "Battle.net". Your computer's settings will have already determined your default server.

  3. Create an Account if you have not done so already.

  4. When you have reached the Message of the Day (MotD) screen, choose "Play Game".

  5. Using the drop down menus, choose a race, and the type of game you want to play.

  6. You can also set your map preferences while here.

  7. When you are ready, choose "Play Game", and Battle.net will search for a suitable game based on your specifications.


Creating Team Ladder Games

To create a Team AMM Game, follow these steps:

NOTE: You must have an active Internet connection.

  1. Start Warcraft III: The Frozen Throne.

  2. Select "Battle.net". Your computer's settings will have already determined your default server.

  3. Create an Account if you have not done so already.

  4. When you've reached the Message of the Day (MotD) screen, choose "Enter Chat".

  5. Select "Channel".

  6. Enter the name of the Channel you wish to create and choose "Join Channel". This will automatically create the Channel if it does not already exist.

  7. Once in the private Channel, choose "Arranged Teams".

  8. Every player in your private Channel will appear on the Battle.net Teammate Invitation screen. Select your Teammates by checking the box next to their name, and choose "Invite". Do NOT choose "Invite" until you have checked all the players you wish to invite. You may have up to three teammates (four players on one team total). To leave this screen, choose "Cancel".
    NOTE: Mutual "friends" will always appear on each other's Invitation screen no matter what Channel they are in.

  9. After you choose "Invite" you will have the option to select map and race preferences, along with which game-type you wish to play.

  10. When your team is ready, choose "Play Game", and Battle.net will look for a suitable team for a match.


Joining Team Ladder Games

To join a Team AMM Game, follow these steps:

NOTE: You must have an active Internet connection.

  1. Start Warcraft III: The Frozen Throne.

  2. Select "Battle.net". Your computer's settings will have already determined your default server.

  3. Create an Account if you have not done so already.

  4. When you have reached the Message of the Day (MotD) screen, choose "Enter Chat".

  5. Choose "Channel".

  6. Type in the name of the private Channel you and your teammates have agreed on, and choose "Join Channel".

  7. When the game's creator invites you, you will have thirty seconds to accept his offer. If you do not reply in thirty seconds, you will automatically decline.

  8. Select your race preference, and wait for the game's creator to start the search process.


Creating Custom Games

To create a Custom Game, follow these steps:

NOTE: You must have an active Internet connection.

  1. Start Warcraft III: The Frozen Throne.

  2. Select "Battle.net". Your computer's settings will have already determined your default server.

  3. Create an Account if you have not done so already.
  4. When you've reached the Message of the Day (MotD) screen, choose "Custom Game".

  5. Choose "Create Game".

  6. Select a game name.

  7. At this point, choose if you would like to host a Private or Public game. Private games will not show up on the list of available games, but anyone you wish to participate will have to know your game's name. Public games are open to all players.

  8. Choose a map.

  9. If you would like to set any of the Advanced Options do so now. More details about Advanced Options are found under Single Player in the Warcraft III Read Me.

  10. Choose "Create Game".

  11. Once you have enough players, and everyone has selected a race, choose "Start Game" to have the game begin.


Joining Custom Games

To join a Custom Game, follow these steps:

NOTE: You must have an active Internet connection.

  1. Start Warcraft III: The Frozen Throne.

  2. Select "Battle.net". Your computer's settings will have already determined your default server.

  3. Create an Account of you have not done so already.

  4. When you have reached the Message of the Day (MotD) screen choose "Custom Game".

  5. At this point, you may select a game from the list of games created by other players, or you can type in a game name if you already know the name. After you type in the game's name, or select a game from the list, choose "Join Game".




Local Area Network Games

Creating Local Area Network (LAN) Games

To create a LAN Game, follow these steps:

NOTE: You must have your computer set up on a network to have friends join your LAN game. You can play LAN games with computer-controlled opponents if you wish.

  1. Start Warcraft III: The Frozen Throne.

  2. Select "Local Area Network".

  3. Enter a "Name" for yourself.

  4. Choose "Create Game".

  5. Choose a map and set any Advanced Options you wish to alter. More details about Advanced Options can be found under Single Player in the Warcraft III Read Me.

  6. Choose "Create Game".

  7. After other players have joined, or you have set computer-controlled opponents, choose "Start Game" to begin.


Joining Local Area Network (LAN) Games

To join a LAN Game, follow these steps:

NOTE: You must have your computer set up on a network to join a LAN game. If you want to play a LAN game with computer-controlled opponents, see Creating Local Area Network Games.

  1. Start Warcraft III: The Frozen Throne.

  2. Select "Local Area Network".

  3. Enter a "Name" for yourself.

  4. Select a game from the list and choose "Join Game".
